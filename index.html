داونلۆدکردنی DNS

سەرەتا پێویستە ئەم دی ئێن ئێسە دابەزێنیت بۆ ئەوەی ڕێگری لە پوچەڵبوونەوەی سێرتیفیکەیت بگریت.

📥 داونلۆدکردنی DNS

⚙️ هەنگاوەکانی جێگیرکردن (Settings):

  1. بڕۆ بۆ سیتینگی ئایفۆنەکەت (Settings).
  2. بچۆ بەشی گشتی (General).
  3. بگەڕێ بۆ VPN & Device Management.
  4. لەنێو DNS، ئەوەی تازە داتگرتووە هەڵیبژێرە.
٢

داونلۆدکردنی ESign

ئێستا ئەپڵیکەیشنی ئیساین دابەزێنە.

📲 داونلۆدکردنی ESign ‏

📖 فێرکاری:

  1. کلیک لەسەر داونلۆد بکە و چاوەڕێ بکە Install بێت.
  2. ئەگەر نامەی "Untrusted Developer" هات، بڕۆ بۆ Settings > General > VPN & Device Management و Trust بکە.
٣

داونلۆدکردنی فایلی سێرڤەر

فایلی تایبەت بە کارپێکردنی سێرتیفیکەیتەکان دابەزێنە.

📄 داونلۆدکردنی فایل

📥 چۆنیەتی داخڵکردن بۆ ناو ئیساین:

  1. فایلەکە دابەزێنە و با لەناو بەرنامەی Files ی ئایفۆنەکەت بێت.
  2. بەرنامەی ESign بکەرەوە.
  3. بچۆ بەشی (File) لە خوارەوە، پاشان سێ خاڵەکەی سەرەوە، Import دابگرە.
  4. فایلە دابەزێنراوەکە هەڵبژێرە و بیکەرەوە.
  5. کلیک لەسەر فایلەکە بکە لەناو ئیساین و بیخە ناو سێرتیفیکەیتەکان (Import to Certificate).
